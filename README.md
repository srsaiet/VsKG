# VsKG
Generador de Licencias para Visual Studio

Solo basta seleccionar la Edición, la cantidad de licencias a generar, pulsar el boton generar, y apareceran las licencias en la parte inferior.
Las licencias son 100% válidas para la activación del producto seleccionado, pero no son válidas para una auditoria ...

Este programa fue creado con propositos educativos, el autor no se hace responsable del mal uso que puedan darle al código

Nota, el archivo pkconfig.xrm-ms es para la versión 2026, si desean una versión anterior solo cambien el archivo por el de la versión que requieran, y lo encuentran en la carpeta de instalación de la versión a generar las licencias.
